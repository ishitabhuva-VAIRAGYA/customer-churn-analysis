# Customer Churn Business Analysis

## Project Overview
Customer churn is a critical problem for subscription-based businesses. 
Acquiring a new customer costs significantly more than retaining an 
existing one. This project analyzes customer churn patterns using data 
analysis techniques to identify the key factors driving customer 
attrition and provide actionable business recommendations.

## Objectives
• Identify key factors influencing customer churn  
• Perform exploratory data analysis and statistical analysis  
• Create visual insights for decision-makers  
• Provide data-driven business recommendations  

## Dataset
Customer Churn Dataset

Rows: 500  
Columns: 4

Features:
- Customer ID
- Monthly Charges
- Tenure
- Churn (Yes/No)

## Analysis Techniques Used
1. Exploratory Data Analysis
2. Correlation Analysis
3. Statistical Hypothesis Testing

## Business Insights
Customers with low tenure and high monthly charges show the highest 
churn probability.

## Business Recommendations
• Offer loyalty rewards for customers beyond 12 months
• Provide discounted plans for high monthly charge customers
• Improve onboarding experience
